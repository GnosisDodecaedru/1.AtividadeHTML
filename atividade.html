Introdução à Linguagem C

1. Estrutura Básica de um Programa em C

Aprendi que todo programa em C começa pela função principal chamada main. Ela é o ponto de entrada do programa.

#include <stdio.h>

int main() {
    return 0;
}

Explicação:

  • #include <stdio.h> → Biblioteca padrão para entrada e saída de dados.
  • int → Tipo de dado da função (inteiro).
  • main() → Função principal do programa.
  • { } → Delimitam o bloco da função.
  • return 0; → Indica que o programa terminou corretamente.

2. Tipos de Dados

Aprendi que em C precisamos definir o tipo da variável antes de usá-la.

  • int → Números inteiros (ex: 10, -5)
  • float → Números decimais (ex: 5.5)
  • char → Um único caractere (ex: 'A')
  • double → Números decimais com mais precisão

3. Variáveis

Aprendi a declarar e utilizar variáveis:

int idade = 18;
float altura = 1.75;
char letra = 'A';

Variáveis servem para armazenar informações na memória do computador.

4. Entrada e Saída de Dados

Também aprendi a usar printf para mostrar informações na tela e scanf para receber dados do usuário.

#include <stdio.h>

int main() {
    int idade;

    printf("Digite sua idade: ");
    scanf("%d", &idade);

    printf("Sua idade é %d", idade);

    return 0;
}
