O'que eu aprendi nessas aulas do Mr. Gostoso



1. Aprendi a criar uma função


EXEMPLO:


int main {

return 0;

}

Neste exemplo, criei uma função chamada "main", do tipo inteiro, "int", que é o tipo de dados que ela retornará ao usuário, abri colchetes "{" para "abrir" a função. "return 0;" no final para retornar um número inteiro, que no caso é o 0, depois fechei os colchetes. E pronto! Uma função que simplesmente retorna o valor zero criada!
